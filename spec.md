# Core Hugo theme spec

This codebase is for a high-quality production-grade core Hugo theme that can be customized and extended later on.

## Requirements

* [x] Homepage should have a metadata item showing last update of the most recent post, if available
* [x] Move favicons to static/icons/
* [x] Move the security files (headers etc.) out, document how to implement them instead
* [x] Make sure the site's name in the footer links to the site
* [x] Fix missing web manifest file
* [x] Make sure that button text is readable on hover; right now it's blue-on-blue
* [x] Dynamic page title
* [x] Full RSS support
* [x] Robots text file
* [x] Open Graph metadata
* [x] Twitter Card metadata
* [x] Structured data (JSON-LD)
* [x] Trailing slash consistency
* [x] Sitemap
* [x] Exactly one H1 tag per page
* [x] Usable without JavaScript
* [x] Accessibility fundamentals (WCAG, semantic HTML, proper landmarks like main/nav/footer, input labels) handled
* [x] Search index that can be used by Fuse or Lunr
